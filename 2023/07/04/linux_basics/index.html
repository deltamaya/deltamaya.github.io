linux基本指令

ls

short for list

ls [path] [options]

打印某个目录（默认是工作目录）下的文件/目录

常用选项：

  • -a 打印隐藏文件
  • -l 打印更多信息（ls -l 可简写为ll）
  • -d 将文件夹像文件一样显示
  • -t 以时间排序
  • -R 列出子目录下的文件

pwd

short for print working directory

打印工作目录

cd

short for change directory

cd [path]

更换工作目录

linux中使用 .表示当前目录， ..表示上级目录， /表示根目录

cd ~切换到用户目录

cd -切换到上一个工作目录

stat

stat [file]

用于展示文件属性(size/access/modify/change/uid/gid)

touch

用于更改文件的access/modify/change时间，也可以用于创建文件

access时间是访问权限，理论上是最常被更改的，但是为了减少磁盘IO设置为了隔一段时间更改一次

modify是修改文件内容的时间

change是修改文件属性的时间（文件属性一般包括权限、大小等）

touch [file] [options]

常用选项:

  • -a 更改access时间，同时也会修改change
  • -t 使用特定的时间，不是当前时间（日期格式：）

mkdir

mkdir [path/name]

在path路径下创建一个name名字的文件夹

常用选项：

  • -p（–parent）递归创建文件夹

rmdir&rm

rm [path/name]

删除文件/目录

常用选项：

  • -r 递归删除
  • -f 强制删除不询问
  • -i 交互删除询问

ps：rmdir只能删除空文件夹

man

short for manual

大名鼎鼎的RTFM

man [index] [command]

常用目录：

  • 1 linux指令页
  • 2 系统调用页
  • 3 c库函数页
  • 8 系统管理指令页

常用选项：

  • -a 从所有目录中查找

cp

short for copy

copy [file] [path]

复制文件file到路径path下

常用选项：

  • -r 递归处理子文件和目录
  • -i 交互询问是否覆盖文件
  • -f 强制拷贝并覆盖

mv

short for move

mv [source/name] [destination/name]

从source路径移动到destination路径

也可以用于文件改名

常用选项与cp一致

cat

cat [path/name]

常用选项：

  • -b 对非空输出行编号
  • -n 对输出的所有行编号
  • -s 不输出多行空行

head&tail

查看文件前/后 n行

head/tail -number [file]

less

常用选项：

  • -i 忽略搜索时的大小写
  • -N 显示每行的行号
  • /字符串：向下搜索“字符串”的功能
  • ?字符串：向上搜索“字符串”的功能
  • n：重复前一个搜索（与 / 或 ? 有关）
  • N：反向重复前一个搜索（与 / 或 ? 有关）
  • q:quit

find

find [path/name] [options]

在文件树中查找文件

常用选项：

  • -name 根据名字查找文件

例如： find /home/deltamaya -name test.c

查找deltamaya家目录下名为test.c的文件

grep

short for global regular expression

grep [options] [target] [file]

常用选项：

  • -i ：忽略大小写的不同，所以大小写视为相同
  • -n ：顺便输出行号
  • -v ：反向选择，亦即显示出没有 ‘搜寻字符串’ 内容的那一行

例如：grep -n fork fork.cc

在fork.cc文件中使用正则表达式搜索fork字符串，并且输出行号

zip&unzip

例子：

zip -r test.zip test.asm test.c test.o

unzip -f test.zip

tar

文件的压缩和解压

常用选项：

  • -c ：建立一个压缩文件的参数指令(create 的意思)；
  • -x ：解开一个压缩文件的参数指令！
  • -t ：查看 tarfile 里面的文件！
  • -z ：是否同时具有 gzip 的属性？亦即是否需要用 gzip 压缩？
  • -j ：是否同时具有 bzip2 的属性？亦即是否需要用 bzip2 压缩？
  • -v ：压缩的过程中显示文件！这个常用，但不建议用在背景执行过程！
  • -f ：使用档名，请留意，在 f 之后要立即接档名喔！不要再加参数！
  • -C ： 解压到指定目录

例子：

tar -zcv test.c test.o -f test.tar.gz

将test.c test.o打包并使用gz压缩为test.tar.gz

tar -xvf test.tar.gz -C ./d1

从test.tar.gz中解压放入d1目录中

date

date +[format]

格式：

  • %H : 小时(00..23)
  • %M : 分钟(00..59)
  • %S : 秒(00..61)
  • %X : 相当于 %H:%M:%S
  • %d : 日 (01..31)
  • %m : 月份 (01..12)
  • %Y : 完整年份 (0000..9999)
  • %F : 相当于 %Y-%m-%d

cal

cal [year]

打印年year的日历

uname

获取当前系统的信息

常用选项：

  • -a 展示全部信息

mount&umount

挂载/卸载文件系统

ifconfig

展示网络信息

top

展示cpu占用

free

展示内存占用

chmod

chmod [options] [expression] [file]

exp可以使用a/u/g/o +/- r/w/x

或者只用三个八进制数字

更改文件访问权限

chgrp

更改文件所属组

umask

查看权限掩码

su

切换用户

rz&sz

文件类型

  • d：文件夹
  • -：普通文件
  • l：软链接（类似Windows的快捷方式）
  • b：块设备文件（例如硬盘、光驱等）
  • p：管道文件
  • c：字符设备文件（例如屏幕等串口设备）
  • s：套接口文件

vim基本使用

插入模式

  • 按「i」切换进入插入模式「insert mode」，按“i”进入插入模式后是从光标当前位置开始输入文件；
  • 按「a」进入插入模式后，是从目前光标所在位置的下一个位置开始输入文字；
  • 按「o」进入插入模式后，是插入新的一行，从行首开始输入文字。
    从插入模式切换为命令模式
    按「ESC」键。

移动光标

  • vim可以直接用键盘上的光标来上下左右移动，但正规的vim是用小写英文字母「h」、「j」、「k」、
    「l」，分别控制光标左、下、上、右移一格
  • 按「G」：移动到文章的最后
  • 按「 $ 」：移动到光标所在行的“行尾”
  • 按「^」：移动到光标所在行的“行首”
  • 按「w」：光标跳到下个字的开头
  • 按「e」：光标跳到下个字的字尾
  • 按「b」：光标回到上个字的开头
  • 按「#l」：光标移到该行的第#个位置，如：5l,56l
  • 按［gg］：进入到文本开始
  • 按［shift＋g］：进入文本末端
  • 按「ctrl」+「b」：屏幕往“后”移动一页
  • 按「ctrl」+「f」：屏幕往“前”移动一页
  • 按「ctrl」+「u」：屏幕往“后”移动半页
  • 按「ctrl」+「d」：屏幕往“前”移动半页

删除文字

「x」：每按一次，删除光标所在位置的一个字符
「#x」：例如，「6x」表示删除光标所在位置的“后面（包含自己在内）”6个字符
「X」：大写的X，每按一次，删除光标所在位置的“前面”一个字符
「#X」：例如，「20X」表示删除光标所在位置的“前面”20个字符
「dd」：删除光标所在行
「#dd」：从光标所在行开始删除#行

复制

「yw」：将光标所在之处到字尾的字符复制到缓冲区中。
「#yw」：复制#个字到缓冲区
「yy」：复制光标所在行到缓冲区。
「#yy」：例如，「6yy」表示拷贝从光标所在的该行“往下数”6行文字。
「p」：将缓冲区内的字符贴到光标所在位置。注意：所有与“y”有关的复制命令都必须与“p”配合才能完
成复制与粘贴功能。

替换

「r」：替换光标所在处的字符。
「R」：替换光标所到之处的字符，直到按下「ESC」键为止。

撤销上一次操作

「u」：如果您误执行一个命令，可以马上按下「u」，回到上一个操作。按多次“u”可以执行多次回
复。
「ctrl + r」: 撤销的恢复

更改

「cw」：更改光标所在处的字到字尾处
「c#w」：例如，「c3w」表示更改3个字

跳至指定的行

「ctrl」+「g」列出光标所在行的行号。
「#G」：例如，「15G」，表示移动光标至文章的第15行行首。
